# Skill Self-Improvement Compatibility Pointer

The canonical self-improvement policy is now the
[`docs/improvements/IMPROVEMENTS_PROTOCOL.md`](../improvements/IMPROVEMENTS_PROTOCOL.md).

For skills, append proposals to
[`docs/improvements/IMPROVEMENTS_SKILLS.md`](../improvements/IMPROVEMENTS_SKILLS.md). Repo-owned
skills still live only in `docs/skills/<name>/SKILL.md`; never edit an adapter or symlink copy.
